Libro Historia PreUN - Proyectia HTML interactivo

Esta versión no incrusta el PDF como imagen. Usa texto extraído y reconstruido, conserva las imágenes principales del PDF original y organiza el material como una aplicación HTML local.

Uso:
1. Abrir index.html en un navegador.
2. En la página inicial, elegir una lectura.
3. En cada pregunta, seleccionar primero una opción.
4. Después de seleccionar, usar el botón 'Ver respuesta correcta' para revisar la clave y la explicación.

Estructura:
- index.html: aplicación interactiva.
- assets/: imágenes conservadas del PDF original.
- index_secuencial_backup.html: copia de la versión secuencial anterior.
